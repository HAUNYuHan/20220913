## 說明何謂案例圖 使用案例 行為者
### 使用案例圖
```
利用圖形表達，以鳥瞰的觀點可快速一窺系統的全貌。界定系統範圍(System Boundary)、找出系統主要(Primary)及支援參與者(Support)，以及系統所需提供的功能與擔任的責任
```
### 使用案例
```
是用來描述某項特定工作，透過使用案例的執行以達成系統所需的功能
```
### 行為者
```
表示使用者與系統互動時所扮演的角色，一個行為者可以參與多個使用案例的執行
```
## 說明使用案例圖 會有哪幾種關係
```
1.一般化Generalization關係
2.包含Include關係
3.擴充Extend關係
```
